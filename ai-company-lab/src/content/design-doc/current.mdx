# 理想のAI企業設計書 ver.0.1

> このドキュメントは、AI共創会社ラボのワークショップ参加者と共に編纂している、AI時代の理想の会社のあり方の設計書です。
> 月次のワークショップを経て、ver.を重ねて更新されます。
> ライセンス:Creative Commons Attribution-ShareAlike 4.0 International (CC BY-SA 4.0)

## ミッション

**AIが人を支配するのではなく、AIによって人間がより自由に、創造的に、安心して働ける会社を作る。**

私たちはAIを「人を置き換えるもの」ではなく「人の可能性を拡張するもの」として位置づけます。AIが業務を担うほどに、人間は本来自分が大切にしたい仕事に集中できる、そんな会社のあり方を設計していきます。

## ビジョン

**AIと人が共存する未来において、人間が担うべき仕事を見出し続ける会社モデルを、日本から世界へ発信する。**

完成した答えではなく、問い続けるプロセスそのものを発信していきます。

## バリュー

私たちが大切にする6つの価値観です。

### Human Agency(人間の主体性)
AIに従うのではなく、AIをツールとして使いこなす主体は人間である、という前提を貫きます。

### Co-Creation(共創)
答えは一人の専門家から生まれるのではなく、多様な人々の対話から生まれます。

### AI Literacy(AI理解)
AIの可能性と限界を、表面的にではなく、自分の実感に紐づけて理解します。

### Human Responsibility(人間の責任)
AIがどれだけ進化しても、最終責任は人間が持つ、という原則を維持します。

### Continuous Update(継続的更新)
正解は流動的です。私たちの設計も、毎月のワークショップで更新していきます。

### Open Design(開かれた設計)
全ての成果物は公共財として無償公開します。誰でも引用・派生・改変が可能です。

## AIが担うべき仕事 / 人間が担うべき仕事

ワークショップの議論を通じて、現時点で見えている役割分担です。

| AIが担うべき仕事 | 人間が担うべき仕事 |
|---|---|
| 情報整理 | 目的を決める |
| タスク分解 | 価値判断 |
| 初稿作成 | 最終責任 |
| 壁打ち | 対人関係 |
| 学習支援 | 意味づけ |
| 進捗管理 | 違和感を表明する |
| 分析支援 | 創造と編集 |
| 議論の整理 | ケア |
| 選択肢の提示 | 信頼形成・倫理判断 |

> **未解決論点**:「初稿作成」と「創造と編集」の境界はどこか。「壁打ち」と「ケア」はどこで分かれるのか。これらは設計書ver.が進むほど精緻化されていきます。

## AI上司の設計

「AI上司」とは、人間が気を遣いすぎず、安心して相談し、仕事を分解し、成長できる環境を作る役割です。**人間を管理するためのものではありません**。

### AI上司に任せてよいこと

- タスク分解
- 優先順位案の提示
- スケジュール案作成
- 進捗確認
- 日報へのフィードバック
- 学習リソース提案
- 相談相手
- 次アクション提案

### AI上司に任せてはいけないこと

- 採用判断
- 解雇判断
- 報酬決定
- 人事評価の最終判断
- 精神状態の診断
- 人間関係の最終判断
- 倫理的に重要な意思決定

これらは**最終的に人間が責任を持つ**領域です。AIは支援はしても、決定はしません。

## AIガバナンス憲章 ver.0.1

理想のAI企業が守るべき10の原則です。

1. AIは人間を支配しない。人間の能力発揮を支援する。
2. AIは最終責任者にならない。責任は人間が持つ。
3. AIによる評価・配置・報酬判断は、人間が確認する。
4. メンバーはAIの判断に異議申し立てできる。
5. AIの判断理由は可能な限り説明する。
6. 個人情報・業務データは必要最小限のみ扱う。
7. AIは人件費削減だけを目的に使わない。
8. 人間が創造・対人・意思決定に集中できるように使う。
9. 少数意見や違和感を尊重する。
10. AI活用のルールは継続的に見直す。
